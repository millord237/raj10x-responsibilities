# Accountability Coach Workspace

This folder contains all files and content related to your **Accountability Coach** agent.

## Folder Structure

- `files/` - General files uploaded by you
- `images/` - Generated or uploaded images
- `videos/` - Generated or uploaded videos
- `summaries/` - AI-generated summaries and insights
- `notes/` - Meeting notes and personal notes
- `exports/` - Exported content (PDFs, reports)
- `chats/` - Chat history with this agent
- `generated/` - All AI-generated content

## Agent Details

- **ID:** accountability-coach
- **Description:** Your personal accountability partner who helps you stay on track with your goals
- **Skills:** Streak tracking, Daily check-ins, Smart scheduling

## How to Use

1. **Upload files** - Drag and drop or use the paperclip icon in chat
2. **View files** - Click on any file in the left panel to view it
3. **Edit files** - Click Edit button to modify text files
4. **Download** - Use the download button to save files locally

## File Types Supported

| Type | Extensions | View | Edit |
|------|------------|------|------|
| Documents | `.md`, `.txt` | Rendered | Yes |
| Code | `.js`, `.ts`, `.py`, `.json` | Syntax highlighted | Yes |
| Images | `.png`, `.jpg`, `.gif`, `.webp` | Preview + Fullscreen | No |
| Videos | `.mp4`, `.webm` | Player | No |
| Audio | `.mp3`, `.wav` | Player | No |
| Data | `.csv`, `.json`, `.yaml` | Formatted | Yes |
| PDF | `.pdf` | External viewer | No |
